# Algorithm-Study!
> Repository for Python and JS Algorithm Study
## Algorithm Language
 ```Python``` ```JavaScript```
## Daily Study Rule
* 하루 2 문제 ```Python```으로 풀이 후 택 1 하여 ```JavaScript```로 풀이
  1. Algorithm-Study 저장소 ```Fork```
  2. 로컬 저장소로 ```clone``` 
  3. 본인 이름으로 `branch` 생성
    * ex) jisu
  4. 작업 후 `commit`, `push`
    * 파일 경로: programmers/주제/본인이름/문제이름   **ex) programmers/stack_queue/jisu/같은숫자는싫어.py**
    * commit 양식: "[Programmers]주제-문제이름"
  5. 본인 계정 github 저장소에서 `PR(Pull Requst)`
## Weekly Meeting
* 매주 수요일(가정) 온라인으로 진행
* 각자 한 주 간 풀이한 문제 중 택 1
  * 간단히 관련 이론 브리핑
  * 본인 문제 풀이방식/코드 설명 -> 코드리뷰
  * 스터디 후 코드 수정하여 `PR`
* 주제 겹치지 않도록 매주 금요일에 문제 및 주제 배분
